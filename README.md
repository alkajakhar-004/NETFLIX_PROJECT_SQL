# PROJECT OVERVIEW
This project analyzes Netflix Movies and TV Shows data using SQL and Power BI. The dataset was loaded from CSV into SQL, insights were extracted using SQL queries, and an interactive dashboard was created in Power BI.

# OBJECTIVE
- Analyze Netflix content trends
- Compare Movies and TV Shows
- Identify top genres
- Find leading content-producing countries
- Study ratings distribution
